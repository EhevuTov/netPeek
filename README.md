## Description

This is a demo of a realtime phone call generator between a console
server and a web client

## Features

- real-time messages
- data grid

## Goals

- login authorization
- msu/sec graph
- filters
- decode message window

## Usage

### Web

    node support/msu_gen.js
    node app.js

### Console

    node support/msu_gen.js
    node support/msu_cap.js
